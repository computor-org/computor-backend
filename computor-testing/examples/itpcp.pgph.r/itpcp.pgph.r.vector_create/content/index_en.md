# Creation of Vectors in R

## Learning Objectives

In this example you will learn how to create vectors in R using various methods.

## Tasks

1. Create simple vectors using `c()`
2. Create sequences using `seq()` and the colon operator
3. Create logarithmic and exponential sequences
4. Perform mathematical operations on vectors

## Expected Variables

After completing this exercise, your script should define the following variables:
- `zero`, `one_vec`, `five_vec`, `irregular` - simple vectors
- `seq_1`, `seq_2`, `seq_3`, `seq_by`, `seq_len` - sequence vectors
- `log_seq`, `log_values`, `exp_seq`, `exp_values` - logarithmic/exponential
- `pow_2`, `log2_pow`, `cumsum_vec`, `factorial_vec` - mathematical operations
