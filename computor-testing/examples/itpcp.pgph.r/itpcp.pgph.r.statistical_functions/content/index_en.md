# Statistical Functions in R

## Learning Objectives

In this example you will learn how to use R's built-in statistical functions.

## Tasks

1. Calculate basic statistics (mean, median, sd, var)
2. Find range and quantiles
3. Calculate correlation and covariance
4. Work with summary statistics

## Expected Variables

After completing this exercise, your script should define the following variables:
- `data_vec` - the data vector
- `data_mean`, `data_median`, `data_sd`, `data_var` - basic statistics
- `data_min`, `data_max`, `data_range`, `data_quantiles`, `data_iqr` - range/quantiles
- `correlation`, `covariance` - correlation/covariance
- `sorted_data`, `unique_count`, `freq_count` - summary statistics
