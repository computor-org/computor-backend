# Control Structures in R

## Learning Objectives

In this example you will learn how to use control structures in R.

## Tasks

1. Use if-else statements for conditional logic
2. Use for loops for iteration
3. Use while loops for conditional iteration
4. Use switch statements for multiple conditions

## Expected Variables

After completing this exercise, your script should define the following variables:
- `if_result`, `nested_if` - if-else results
- `for_sum`, `for_product` - for loop results
- `while_count`, `while_sum` - while loop results
- `switch_result` - switch statement result
