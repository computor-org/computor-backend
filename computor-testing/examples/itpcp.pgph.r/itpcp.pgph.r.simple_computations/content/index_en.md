# Simple Computations in R

## Learning Objectives

In this example you will learn how to perform simple computations using operators and predefined functions in R.

## Tasks

1. Create variables for basic arithmetic operations
2. Use mathematical functions (sqrt, exp, log, sin, cos)
3. Work with vectors and element-wise operations
4. Understand R's numeric types

## Expected Variables

After completing this exercise, your script should define the following variables:
- `a`, `b`, `c` - basic numeric variables
- `sum_ab`, `diff_ab`, `prod_ab`, `div_ab` - arithmetic results
- `sqrt_b`, `exp_a`, `log_b` - mathematical function results
- `sin_val`, `cos_val` - trigonometric results
- `vec`, `vec_squared` - vector operations
