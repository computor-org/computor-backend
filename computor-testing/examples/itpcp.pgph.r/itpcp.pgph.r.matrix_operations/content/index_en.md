# Matrix Operations in R

## Learning Objectives

In this example you will learn how to create and manipulate matrices in R.

## Tasks

1. Create matrices using `matrix()`
2. Perform matrix transposition with `t()`
3. Perform matrix multiplication with `%*%`
4. Extract rows, columns, and diagonals

## Expected Variables

After completing this exercise, your script should define the following variables:
- `mat_a`, `mat_b` - basic matrices
- `mat_t` - transposed matrix
- `mat_mult`, `elem_mult` - matrix and element-wise multiplication
- `row_1`, `col_2`, `diag_mat` - extracted elements
- `identity_mat`, `zero_mat` - special matrices
