# Data Frames in R

## Learning Objectives

In this example you will learn how to work with data frames in R.

## Tasks

1. Create data frames using `data.frame()`
2. Access columns using `$` and `[[]]` notation
3. Filter rows based on conditions
4. Perform basic operations on data frames

## Expected Variables

After completing this exercise, your script should define the following variables:
- `df` - the main data frame
- `df_names`, `df_ages` - extracted columns
- `df_nrow`, `df_ncol` - dimensions
- `df_filtered` - filtered data frame
- `df_summary` - summary information
