# String Operations in R

## Learning Objectives

In this example you will learn how to work with strings in R.

## Tasks

1. Concatenate strings using `paste()` and `paste0()`
2. Extract substrings using `substr()`
3. Convert case using `toupper()` and `tolower()`
4. Use pattern matching with `grep()` and `grepl()`

## Expected Variables

After completing this exercise, your script should define the following variables:
- `str1`, `str2` - basic strings
- `concat_str`, `paste0_str` - concatenated strings
- `substr_result` - substring extraction
- `upper_str`, `lower_str` - case conversion
- `grep_result`, `grepl_result` - pattern matching results
