# Functions in R

## Learning Objectives

In this example you will learn how to define and use functions in R.

## Tasks

1. Define basic functions with `function()`
2. Use default arguments in functions
3. Create recursive functions
4. Write functions that operate on vectors

## Expected Variables

After completing this exercise, your script should define the following variables:
- `square_result`, `cube_result`, `add_result` - basic function results
- `greet_default`, `greet_custom` - functions with default arguments
- `factorial_result`, `fibonacci_result` - recursive function results
- `is_prime_7`, `is_prime_8` - prime check results
- `vec_norm`, `vec_normalized` - vector function results
