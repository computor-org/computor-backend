# Erzeugen von Zufallszahlen

## Einleitung

In diesem Beispiel bekommen Sie eine Einführung in die Erzeugung von Zufallszahlen mit Python. Da es extrem wichtig ist, dass man sich in der Python-Dokumentation zurechtfindet, werden für die
folgende Teilaufgabe nicht alle Befehle angeführt.


## Aufgabe

1. Erzeugen Sie im File `vector_random.py` folgende Vektoren mit Zufallszahlen:

| Variable      |Wert|
|---------------|:-----------------------------------------|
| `rand1`       | 10 gleichverteilte Zufallszahlen im halboffenen Intervall $[0,1)$ |
| `rand2`       | 20 normalverteilte Zufallszahlen (Mittelwert 0, Standardabweichung 1) |
| `rand_normal` | 20 normalverteilte Zufallszahlen (Mittelwert 5, Standardabweichung 0.2) |
| `i_min`       | 5 |
| `i_max`       | 15 |
| `randint1`    | 30 gleichverteilte ganzzahlige Zufallszahlen im geschlossenen Intervall `[i_min,i_max]` |
